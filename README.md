# devops_kkit
django2.1写的自动化运维系统
Ssh二次开发，salstack二次开发，MQ,mysql,websocket,redis等等。
核心功能：
1、	操作日志留存、审计、分析。
2、	动态分配临时账号，对服务器等资源进行临时授权。
3、	内部邮件系统。
4、	Ssh，saltstack，web界面三种方式的控制操控渠道。
5、	内部通信及沟通机制。
6、	容器管理（没开发完成）。
7、	物理机、虚拟机等底层资源日志分析及审计。
8、	定时任务。
9、	管理员控制面板。
